a) Clasificación

La clasificación consiste en asignar una categoría a un conjunto de datos en función de sus características. Se utiliza ampliamente en escenarios donde se requiere tomar decisiones basadas en etiquetas predefinidas.

Ejemplo:
  • Clasificar clientes como "alto valor" o "bajo valor".
  • Detección de fraude.
Ejemplo conceptual en Python
from sklearn.tree import DecisionTreeClassifier # Datos de ejemplo X = [[25, 3000], [40, 8000], [35, 5000]] y = [0, 1, 1] # 0 = bajo valor, 1 = alto valor model = DecisionTreeClassifier() model.fit(X, y) predicción = model.predict([[30, 6000]]) print(prediccion)

Este modelo permite clasificar un nuevo cliente según sus características.

b) Clustering (agrupamiento)

El clustering agrupa datos en función de su similitud, sin necesidad de etiquetas previas. Es útil para segmentar clientes o identificar patrones en grandes volúmenes de datos.

Ejemplo:
  • Segmentación de clientes.
  • Agrupación de productos.

c) Regresión

La regresión se utiliza para predecir valores numéricos continuos. Es una de las técnicas más utilizadas en análisis predictivo.

Ejemplo:
  • Predicción de ventas.
  • Estimación de ingresos.
Ejemplo simple de regresión
from sklearn.linear_model import LinearRegression X = [[1], [2], [3], [4]] y = [100, 120, 140, 160] model = LinearRegression() model.fit(X, y) print(model.predict([[5]]))

Este modelo permite estimar un valor futuro basado en una tendencia.

d) Reglas de asociación

Las reglas de asociación permiten identificar relaciones entre variables en grandes conjuntos de datos.

Ejemplo clásico:
  • Análisis de canasta de mercado (market basket analysis).
“Los clientes que compran pan también compran leche”.
