Glosario de la unidad

Análisis de datos

Proceso de inspeccionar, transformar y modelar datos con el objetivo de descubrir información útil, identificar patrones y apoyar la toma de decisiones dentro de una organización.

Cuadro de mando (Dashboard)

Herramienta visual que presenta indicadores clave de desempeño (KPIs) mediante gráficos, tablas y métricas que permiten monitorear el estado de procesos o resultados del negocio de manera rápida y comprensible.

Indicadores clave de desempeño (KPI – Key Performance Indicator)

Métricas utilizadas para medir el desempeño de procesos, objetivos o estrategias dentro de una organización. Permiten evaluar si las metas establecidas están siendo alcanzadas.

Minería de datos (Data Mining)

Conjunto de técnicas analíticas utilizadas para explorar grandes volúmenes de datos y descubrir patrones ocultos, relaciones o tendencias que pueden aportar valor al análisis empresarial.

OLAP (Online Analytical Processing)

Tecnología que permite analizar datos desde múltiples dimensiones, facilitando consultas complejas, agregaciones y análisis comparativos dentro de sistemas de inteligencia de negocios.

Reporte analítico

Documento o visualización que presenta información estructurada derivada del análisis de datos, permitiendo interpretar resultados y apoyar procesos de toma de decisiones.

Tendencia

Patrón o comportamiento observable en los datos a lo largo del tiempo que permite identificar cambios, crecimiento o disminución en determinadas variables analizadas.

Toma de decisiones basada en datos (Data-Driven Decision Making)

Enfoque organizacional que utiliza el análisis de datos y evidencia cuantitativa para orientar decisiones estratégicas, operativas o tácticas dentro de una empresa.

Visual Analytics

Disciplina que combina análisis de datos, visualización interactiva y herramientas tecnológicas para facilitar la exploración de grandes volúmenes de información y mejorar la comprensión de los resultados analíticos.

Visualización de Datos (Data Visualization)

Representación gráfica de datos mediante gráficos, diagramas, mapas o tablas dinámicas que facilitan la interpretación de información compleja y permiten identificar tendencias o patrones relevantes.
